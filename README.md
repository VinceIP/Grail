# GRAIL
### [Game Boy Reverse-engineering AI Lab]

## Overview

GRAIL is an experimental project that provides a lightweight set of command line tools and preconfigured AI agents
to assist in mapping, documenting, and reverse engineering of disassembled Game Boy code.
